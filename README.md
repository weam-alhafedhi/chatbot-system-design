# Chatbot System Design

## Overview
This project was completed as part of a Software Engineering course at Taibah University.

The project focuses on designing a chatbot system, including system architecture, UML diagrams, and software requirements. The project was completed as a group project following software engineering principles.

## Project Components
- System requirements specification
- UML diagrams (Use Case, Class Diagram, Sequence Diagram)
- System architecture design
- Project documentation

## Project Type
Software Engineering Course Project (Group Project)
